# 内容

<a name ="1">一首诗（一）</a>



风花雪月的日子

只有在诗里

才会长久

那不经意间的相逢

总是在热情淡退后

才变得珍惜

用一双饱满泪水的眼，回望

一幕幕过去

我不忍直视前方

是我还满怀深情的望着过去

我只望过这一次



 我用泪书的诗已模糊

我用思摹的景已残度

时光磨了一切

我也宽慰自己说：

我只是没有浅唱出最华美的诗

勾出最匀称的句子

绘出最精巧的词儿

却仍能在一个最美的季节

爱过一个最美的你





<a name ="2">一首诗（二）</a>



诗人总会写好多好多的诗

诗里总是蕴涵着好多情



素未相识的人儿

突如地闯入生活

留下只言片语就匆匆离开

我忙着追逐星月

直到停下，回想着



原来我们依偎地那么近

都让我错过

哪怕

留下一句问候的话儿



爱总是那样的调皮

跑来跑去

 消失不见了

徒留一颗孤寂心

我疯狂地

寻找，寻找，寻找

我遇到过很多人

却再也没见过一个如你般的你





<a name ="3">一首诗（三）</a>



匆匆走过的路却又一次又一次地回眸思绪留恋

唱过的歌都淋在雨里化度成跳动的水丝

读过的故事一遍又一遍在眼眶打转

转着转着成了眼前的现实



追了好多得不到的东西

却终于失去了青春和纯朴的梦



某人说

青春是在筋疲力竭的时候才算度过

那我的青春还长着呢！



我一个一个街道的寻你

试图唤醒一直幻想于梦境中的我

当我做完这惊悚的梦后

醒来

洗干眼角的泪渍

重新寻你



诗尤短

情仍长





<a name ="4">一首诗（四）</a>



当天上乌云密布

当雨从遥远的地方来到脚下

当两个相爱的人

却被这雨隔断

相逢总是如此的简单

离别却又似乎比相逢还简单

相逢与离别之间的徘徊

犹如在雨间穿行

伴着寻觅，伴着抽泣



爱是一种难舍难分的情

世间可以分为两种情感

爱与不爱

我们永远都不会像情书里那般美好

但我们可以选择爱下去

或者放弃去爱对方

总会有一种相逢最终变成离别



我一直这样想

你我的离别就像树叶落到树下



我痴迷地写下一首又一首的诗

直至那不期而遇的相逢

变成了无情的分别

我从不悔过

在那时那地那景遇过那个你





<a name ="5">一首诗（五）</a>



走着走着，我们彼此越来越远

走着走着，我们彼此不会相见

走着走着，我们彼此天各一方

走着走着，笑声变成哭声

走着走着，怨恨变成了思念

上一世的相见

成就了这一世的相恋

我们只是走着走着

彼此不曾回头，脚步放慢

我们只是走着走着

彼此相隔尺间，却心偏地远

走着走着，一切都在瞬间改变

写过的诗终将化作灰烬与倦谈

那爱过的人只能走着走着而疏远

我们在走着的路上错过太多

风景，还有人

世俗的教育告诉我们

莫停下

停下就是堕落与颓靡

但有人说

不要紧盯着这个季节，错过了今冬

好多人走着走着离开了

什么都不会留下

因为走着

并不意味着前进或成长

大多数时候

我们都是在徒劳地走着，走着

并不知道方向



我终于明白

我要时常停下

才不会忘记当时的出发

是为了更好地相见吗？

世上有那么多的路

我只想走你曾走过的

并驻足留恋

然后，停止想念





<a name ="6">一首诗（六）</a>



突然想写一首睡意朦胧的诗

对

我就是睡意朦胧的望着世界的



我想我总会有一天明白

我不曾看清的

也终是无趣



睡意朦胧的诗

就像睡意朦胧的恋爱

一辈子都快过去了

才发现一辈子都快过去了



我望着你很久

拂去睡意，清擦泪水

戴上眼镜

依旧睡意朦胧的望着

那个不曾觉察的你



世间的爱有很多种

那些轰轰烈烈陪你一生的是爱

那个睡意朦胧的在远处望你的也是爱



你会读过好多的诗

我却只望过一个你





<a name ="7">一首诗（七）</a>

此生，我只做过两件事

其一是去爱你

其二是还在爱你





<a name ="8">一首诗（八）</a>



雨停了

那么

我能说一声

我爱你吗？





<a name ="9">一首诗（九）</a>



我们能陪伴吗

我们终能陪伴



从未想象过新的火花

重新燃起

就让那些新旧交替从我眼前消失

我只想静静

陪伴一个人走完一生

一个人就好了



写好又删掉的诗

不是好诗

那些撤回的话语里

总是埋藏了许多眼睛里流不出的泪

我们总是会

在人生最美时错过相逢

又终在要说放弃时相恋

那等不起的缘分

还是会耗尽的

那总以为的机缘巧合

还是会碰巧错过



悔过逃脱不了的相逢

却不会后悔

一时糊涂的爱过





<a name ="10">一首诗（十）</a>



爱从不在虚幻中苟存



人一生可以犯很多错

有一些是永远都不会更正的

望着今晨的月

又总会期待今夜的月

而我只期待过一次



写倦了的诗

就像再也维持不下去的爱

我找寻寻找我落在什么地方

高俊山峰爬了好久

还好记得当初的出发



当初

我只是寻找

一份带有你味道的爱



我还会继续用余生做完另一件事

说一声你好

而不说再见





<a name ="11">一首诗（十一）</a>
